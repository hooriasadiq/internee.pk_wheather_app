# 🌤️ Atmosphere - Modern Weather Dashboard

An elegant, minimal, and responsive Weather Web Application built with native web technologies. Featuring a stunning **Glassmorphic UI**, this application delivers real-time weather analytics with dynamic visual adaptations based on atmospheric conditions.

---

## ✨ Features

* 🔍 **Global City Search:** Instantly fetch live weather data for any city worldwide.
* 📊 **Real-time Dashboard:** Displays precise temperature, humidity levels, and wind speed.
* 🔄 **Dynamic Backgrounds (Bonus):** The entire application theme and background seamlessly morph to match the current weather state (Clear, Clouds, Rain, Snow, Thunderstorm).
* 📍 **Geolocation Support (Bonus):** Built-in Geolocation API integration allows users to discover local weather with a single click.
* 🤖 **Intelligent Error Handling:** Gracefully handles invalid inputs or API failures with beautiful custom alerts.
* 📱 **Fully Responsive:** Tailored for a premium experience across both mobile screens and desktop monitors.

---

## 🛠️ Tech Stack & Concepts Explored

* **Frontend HTML5:** Semantic structures.
* **CSS3 Custom Variables & Glassmorphism:** Implements modern `backdrop-filter` aesthetics and responsive CSS Grids.
* **JavaScript (ES6+):** Utilizes the Native **Fetch API** for asynchronous REST API calls.
* **Third-Party Integration:** Powered by the open-source **OpenWeatherMap API**.
* **Iconography:** Rendered via **FontAwesome v6** for sharp vector-based representations.
